nwiki
=====

Command line client for the [Nostr](https://github.com/fiatjaf/nostr) wiki.

## Installation

Compile with `go install github.com/fiatjaf/nwiki@latest` or [download a binary](https://github.com/fiatjaf/nwiki/releases).

## Usage

```
nwiki

Usage:
  nwiki <article>
  nwiki edit <article>
```
